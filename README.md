# Rule-Based-Expert-System

RULE-BASED EXPERT SYSTEM
=======================
Project: Tech Support Diagnostic System

IMPLEMENTATION:
- Custom forward-chaining engine (no external libraries)
- Rules stored as IF-THEN conditions
- Multi-step inference with chaining
- Complete reasoning path logging

FEATURES:
1. Pattern matching from user input
2. Rule-based diagnosis
3. Forward chaining inference
4. Action recommendations
5. Transparent reasoning path

HOW IT WORKS:
1. User describes problem
2. System extracts facts from keywords
3. Forward chaining applies rules
4. New facts trigger more rules
5. Final diagnosis and actions shown

TO RUN:
python expert_system.py
